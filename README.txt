1) SgCarMart Scrapper
- Scrapes all used car dealers information from https://www.sgcarmart.com/used_cars/index.php 
- Dealership Name, Car Name, Price, Decpreciation, Reg Date, Eng Cap, Mileage, Veh Type, Status
- Stores information into .csv file

2) FanFic Scrapper
- Scrapper for module CZ4034 Information Retrieval
- Scrapes the first page (25 stories) of every movie within the https://www.fanfiction.net/movie/
- Stores in .json format
